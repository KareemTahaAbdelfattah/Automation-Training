# Automation-Training

This repository contains my session-by-session assignments for automation testing using **Selenium WebDriver** and **TestNG**.

## About the Repository
I created this repository to document my progress in learning automation testing and to store all practice tasks, exercises, and assignments in one place.  
The projects here focus on core automation concepts such as:

- Selenium WebDriver basics
- TestNG annotations and assertions
- Locators and element interaction
- Waits and synchronization
- Test case structure and execution
- UI automation practice
- Framework setup and organization

## Purpose
The main goal of this repository is to track my automation testing journey, improve my hands-on skills, and build a collection of practical examples that can be reviewed and expanded over time.

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- IntelliJ IDEA
- Git and GitHub

## Repository Contents
Each folder or package may contain assignments from different training sessions, including:
- Practice tests
- Automation tasks
- Demo scripts
- Mini framework exercises
- Notes and experiment code

## How to Use
1. Clone the repository.
2. Open the project in IntelliJ IDEA or another Java IDE.
3. Run the TestNG test classes from the IDE or using Maven.

This repository is mainly for learning and practice purposes.  
Code structure and implementations may change as I continue improving my automation skills.
